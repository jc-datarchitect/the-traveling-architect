## Overview

**The Traveling Architect** is a specialized recommendation system and route optimizer designed for architects, urban planners, and enthusiasts. By combining spatial analysis with data science, this project transforms architectural exploration into a streamlined, logic-based journey.

As an architect transitioning into Data Science and Business Intelligence, I built this application to solve a recurring problem: how to efficiently organize architectural visits in a foreign city without missing key landmarks, while maintaining a coherent route.

The project combines:

- geospatial data processing
- architectural landmark curation
- route optimization algorithms
- interactive monographic discovery
- minimalist interface design

---

## Project Structure

The project follows a modular architecture, designed for scalability and deployment on Streamlit Cloud:

```text
the-traveling-architect/
├── app.py                  # Main Streamlit UI
├── engine.py               # Route optimization and filtering logic
├── tta_destinations.json   # Datasets of architectural landmarks
├── requirements.txt        # Dependency specifications
└── README.md               # Project documentation
```
